# Documentació del projecte

a partir de dades RNA-seq del projecte TCGA-COAD.

- **[Requisits](requirements.md)** — requisits funcionals i criteris d'acceptació
- **[KPIs](KPI_CHECKLIST.md)** — checklist mesurable de progrés
