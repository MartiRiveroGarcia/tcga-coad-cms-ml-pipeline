# TFG – Pipeline reproduïble (benchmark models)

## Quickstart (placeholder)
1) Crear entorn (conda) amb `environment.yml`
2) Executar pipeline (pendent)

## Estructura
- `scripts/`: execució del pipeline
- `src/`: mòduls reutilitzables
- `notebooks/`: exploració i validació
- `docs/`: documentació (pipeline, reproductibilitat, KPIs)
- `data/manifests/`: manifests/metadata (sense dades grans)
- `results/`: sortides de cada run
